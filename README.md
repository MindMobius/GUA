# GUA · 个人宇宙常量模型推演

一个本地的不确定性归一化装置：你只输入极少信息，系统在你的浏览器里生成并逐步收敛一个“本机宇宙常量模型”，再用它把当下问题压缩成一个可复算的 Ω 与一个清晰的 Score。它不承诺对应现实世界，只承诺对应“你的宇宙”。

## 这是什么

- 一个偏“终端仪式感”的交互：过程要可感知，结果要克制
- 一个偏“可复现”的推演：同一设备/同一输入下，结果稳定或近似稳定
- 一个偏“会学习”的本机模型：推演次数与反馈会改变后续的偏好与收敛速度

## 这不是什么

- 不是现实因果预测器，不提供建议或承诺
- 不是长篇解释型产品：解释只在“解码”阶段出现，结果阶段只展示最少信息

## 使用方式（四步）

- 输入：目标/问题、参考时间、可选标识（严格三项）
- 推演：展示可追溯的推演过程（传统块/现代块），但不把计算细节变成阅读负担
- 归一：把推演结构数值化，生成可复算的“代入等式 = Ω”
- 解码（可选）：可直接对问题进行 AI 推演，或把归一结果解码成更可用的文字（问题解答 + 模型启示）

## 输出是什么

- Ω：以“代入等式 = Ω”的形式呈现，强调可复算与一致性
- Score：0–100 的强度/一致性指标，用于观察趋势（不是吉凶承诺）
- signature：可选短签名，用于标识一次推演的指纹

## 信号来源与隐私

- 默认全部在本地完成：输入、本地观测、推演过程、本机模型、历史记录都保存在你的浏览器里
- 本地观测分两类：被动观测（设备/系统/网络/偏好等）与可选增强观测（需要授权）
- AI 解码是可选能力：只有在你使用解码并配置模型服务时，解码内容才会被发送到你选择的第三方模型

## 结果观（必须贯彻）

- 输出要短：给结论与可复算形式，不讲大道理
- 趋势要真：允许正/负/中性/混合，不强行“总是积极”
- 主观能动性优先：趋势不等于命运，模型只提供“你此刻的宇宙法则”，路径仍可被改写
